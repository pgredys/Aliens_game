# Aliens_game
Quick project based on popular game concept created for fun and education purpose 

## Installation 
```git
git clone https://github.com/pgredys/Aliens_game.git
cd Aliens_game
pip install -r requirements.txt
cd src 
```
### Run game
```python
python alien_invasion.py
```
